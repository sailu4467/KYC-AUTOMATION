
# 🆔 KYC Automation Using ServiceNow

A **ServiceNow-based KYC (Know Your Customer) Automation System** that simplifies and automates the customer verification process. The system manages KYC requests, document verification, approvals, and notifications, reducing manual effort and improving efficiency.

## 🚀 Features
- Online KYC request submission
- Document upload and verification
- Automated approval workflow
- Status tracking for KYC requests
- Email notifications
- Secure customer record management

## 🛠️ Technologies Used
- ServiceNow
- Service Catalog
- Flow Designer
- Business Rules
- Client Scripts
- UI Policies
- Notifications

## 📖 How It Works
1. Customers submit a KYC request through the ServiceNow portal.
2. Required documents are uploaded for verification.
3. The request is routed through an approval workflow.
4. Verification is completed by the assigned team.
5. Customers receive updates on the status of their KYC request.
6. Approved KYC records are securely stored in ServiceNow.
